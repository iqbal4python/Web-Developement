Core Web Vitals

Cumulative Layout Shift (CLS)

Measures visual stability — unexpected layout movement while loading.

Parameter:

  • Good: ≤ 0.1
  • Needs improvement: 0.1–0.25
  • Poor: > 0.25

Largest Contentful Paint (LCP)

Measures loading performance — how long it takes for the largest visible element to render.

Parameter:

  • Good: ≤ 2.5 seconds
  • Needs improvement: 2.5–4.0 seconds
  • Poor: > 4.0 seconds

Interaction to Next Paint (INP)[previously: First Input Delay (FID)]

Measures responsiveness — how quickly the page responds to user interactions (click, tap, keyboard).

Parameter:

  • Good: ≤ 200 ms
  • Needs improvement: 200–500 ms
  • Poor: > 500 ms
